Custom Control Content With AutoResize The content of the Select control can be customized using a custom template which needs to be referenced in the [controlTemplate] input. Select provides the following context values to the template:
  • Text value of selected option - implicit
  • Reference to selected option - selected
Selected option as a context to custom template. Select In Mobile Mode To use Select in mobile mode:
  • import SelectMobileModule
  • Set [mobile] property to true
  • Set [mobileConfig] with MobileModeConfig configuration object to customize the look of the control
Select Mobile Mode can be configured globally by providing MOBILE_MODE_CONFIG token. Check Global Configuration page for more information. DataSource Input [dataSource] has this type type FdpSelectDataSource<T> = SelectDataSource<T> | Observable<T[]> | T[]; Select Max Height The Select component allows you to define the [maxHeight]. It can be useful when there are a lot of options given that the select body is be scrollable. By default the maximum height of the Select body is 45% of the browser's inner window height. Select In A Reactive Form The Select component can be used inside Angular reactive forms.
